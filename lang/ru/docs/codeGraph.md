# CodeGraph

[URL репозитория](https://github.com/colbymchenry/codegraph)

Описание: MCP для индексации файлов в виде графа. После запуска инструмента снижается потребление токенов в ИИ агенте на поиск информации по файлам. Индекс файлов располагается локально в проекте, это также ускоряет поиск в самом ИИ.

Установка глобальная:

```bash
npm install -g @colbymchenry/codegraph
codegraph install --target=codex --location=global --yes
# Или
codegraph install --target=claude --location=global --yes
```

Инициализация MCP в проекте. В папке с проектом нужно выполнить команду

```bash
codegraph init -i
```

После чего прописать в файл AGENTS.md [следующий промт](../examples/codeGraph/agents.md)

Перезапусти Codex и в новой сессии выполни /mcp: должен отображаться сервер codegraph.
На все вопросы ответь Always allow

Для обновления индекса используй команду, но при работе с агентом это происходит автоматически и чаще всего не требуется.

```bash
codegraph sync
```

Для финальной проверки Codex после его перезапуска можно написать ему следующий промт

```
В проекте выполнена настройка MCP CodeGraph для Codex необходимо проверить, что ты теперь работаешь с этим MCP
```

В ответ придёт подтверждение. Если нет, можно попросить ИИ провести диагностику чего именно не хватает для завершения настройки?
