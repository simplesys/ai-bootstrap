# ai-bootstrap

Этот репозиторий содержит базовую структуру для подготовки окружения разработчика к эффективной работе с ИИ-агентами.

## Перевод документации на другие языки

[English](../../README.md)

Для перевода русской документации на английский язык в проекте существует скилл для codex `updocs` достаточно в консоли Codex написать `updocs` и ИИ проверит изменения в папке `lang/ru`, выполнит перевод и обновит английскую версию файлов.

Если же написать сообщение таким образом `updocs --en` тогда произойдёт обратный процесс. Изменения в английской документации будут переведены на русский и обновиться документация в `lang/ru`

## Структура Репозитория

- `AGENTS.md` - основные инструкции для Codex и других ИИ-агентов.
- `CLAUDE.md` - симлинк на `AGENTS.md` для ИИ от Anthropic.
- `SKILLS.md` - локальные повторяемые навыки проекта, включая `/update-docs`.
- `ai-docs/` - инструкции для ИИ и документация репозитория.
- `docs/` - дополнительная документация репозитория.
- `scripts/` - установочные и настроечные скрипты.
- `examples/` - примеры конфигурационных файлов.
- `lang/` - переводы документации проекта на другие языки
