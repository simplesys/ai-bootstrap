# ai-bootstrap

Этот репозиторий содержит базовую структуру для подготовки окружения разработчика к эффективной работе с ИИ-агентами.

Русская версия README используется как основной источник содержания: владелец репозитория описывает изменения на русском языке, а ИИ-агенты должны самостоятельно обновлять и дополнять английскую версию [README.md](README.md) на основе этих изменений.

## Структура Репозитория

- `AGENTS.md` - основные инструкции для Codex и других ИИ-агентов.
- `CLAUDE.md` - симлинк на `AGENTS.md` для инструментов, которые ожидают файл `CLAUDE.md`.
- `docs/` - инструкции для ИИ и документация репозитория.
- `scripts/` - установочные и настроечные скрипты.
- `examples/` - примеры конфигурационных файлов.

## Правило Документации

Если меняется русская версия README, английскую версию нужно обновлять в том же изменении, сохраняя смысл и структуру.
